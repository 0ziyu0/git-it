關卡內容：

幫專案新增協作者。

寫程式邊社交

GitHub 最棒的事情之一就是和人一起協作了，GitHub 把不管什麼時間，都能和來自世界各地的人一起工作變得超簡單。

協作者也是 GitHub 使用者，他們可以編輯屬於別人的 repository。專案要新增協作者，先到 repository 的 GitHub 頁面，按下右邊側欄的 'Settings'（板手和螺絲起子的圖示）。選擇 'Collaborators' 分頁。選擇小隊或建立新小隊。在右上角的文字框輸入協作者的帳號，按 Enter 或是用滑鼠按一下就完成了喔。

步驟： 嗨！專案機器人！

去第六關 fork 出來的 Patchwork repository 的 GitHub 頁面，新增一位協作者：'reporobot'。網址看起來會像是下面這樣，但 'YOURUSERNAME' 會換成你的帳號哦。

github.com/YOURUSERNAME/patchwork/settings/collaboration

確認你完成了所有的步驟：

git-it verify

前往下一關：

git-it

撇步

  • 從 remote 收取更新回來
    • $ git pull <REMOTENAME> <BRANCH>
  • 複製 repository 到自己的電腦裡
    • $ git clone <URL>
  • 新增 remote 連結
    • $ git remote add <REMOTENAME> <URL>
  • 檢視現有 remote
    • $ git remote -v
