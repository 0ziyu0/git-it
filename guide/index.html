Git-it
This app runs in your – Terminal Git-it or Bash (Windows) and has challenges for learning Git and GitHub. It uses the real terminal (so you'll be learning the awesome command line), it uses real Git and GitHub which means when you finish all of the challenges you'll have real repositories on your GitHub account and green squares on your contribution chart.

TOC

  1. Basic command line such as: cd, mkdir
  2. Creating a local repository
  3. Checking status, adding and committing changes
  4. Creating remote repositories on GitHub.com
  5. Forking repositories
  6. Connecting local repositories to remote ones on GitHub.com
  7. Pushing and pulling to sync changes to GitHub.com
  8. Creating branches for features/changes
  9. Adding and working with a collaborator (Hi, @reporobot!)
  10. Creating a pull request
  11. Merging and deleting branches
